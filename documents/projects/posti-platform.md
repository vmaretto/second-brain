# pOsti — Piattaforma Dati Agroalimentari

La piattaforma madre di tutti i progetti. Raccoglie, elabora e gestisce dati per l'intero ecosistema.

## Cos'è

pOsti è una piattaforma per:
- **Acquisizione** dati di filiera agroalimentare
- **Elaborazione** e standardizzazione
- **Notarizzazione** su blockchain
- **Visualizzazione** e reporting

## Caratteristiche

### Data Pipeline
- Raccolta dati da fonti eterogenee (IoT, ERP, documenti)
- Normalizzazione formati
- Validazione e quality check
- Storage strutturato

### Blockchain
- Notarizzazione hash su blockchain pubblica
- Audit trail immutabile
- Certificazione temporale
- Interoperabilità con standard EU

### API & Integrazioni
- REST API per partner
- Integrazione con sistemi EU (EFL, SWITCH)
- Export in formati standard
- Dashboard per stakeholder

## Progetti che usano pOsti

| Progetto | Uso |
|----------|-----|
| **SWITCH** | Data Lake, Smart Counter |
| **EFL/WWF** | Tracciabilità blockchain |
